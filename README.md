# 📱 Dialer App

## 🚀 Overview

The Dialer App is a simple Android application that allows users to enter a phone number and make a call using the device's dialer. It demonstrates the use of Intents for phone actions.

---

## ✨ Features

* 📞 Enter phone number
* 📲 Open dialer with entered number
* ☎️ Direct call functionality
* 🧹 Clear input field
* 🎨 Simple and user-friendly UI

---

## 🛠 Tech Stack

* Java
* XML
* Android Studio
* Android Intents

## ⚙️ How It Works

1. User enters phone number
2. Clicks "Dial" → opens phone dialer
3. Clicks "Call" → initiates direct call

---

## ⚙️ How to Run

1. Clone the repository
2. Open in Android Studio
3. Add CALL_PHONE permission in Manifest
4. Run on Emulator or Device
